# Smart Inventory Optimizer (Система умного распределения ресурсов склада)

## Описание проекта (Цель и задача)
Данный проект представляет собой RESTful API сервис для управления складскими запасами. 
Главная решаемая проблема — заморозка бюджета в избыточных запасах или, наоборот, нехватка критически важных товаров (Stockout).

Сервис не просто выполняет CRUD-операции, но и решает **бизнес-задачу приоритизации закупок**. На основе текущих остатков, минимального требуемого запаса, цены и "фактора риска" категории, встроенный алгоритм (жадный алгоритм рюкзака) автоматически формирует список товаров к закупке, укладываясь в заданный бюджет пользователя.

## Стек технологий
* **Backend:** FastAPI, Python 3.10+
* **Database:** SQLite + SQLAlchemy 2.0
* **Аутентификация:** JWT (JSON Web Tokens), Passlib (Bcrypt)
* **Тестирование:** Pytest, HTTPX, pytest-cov
* **Линтинг:** Pylint, ruff

## Инструкция по локальному запуску

1. Склонируйте репозиторий:
```bash
git clone git@github.com:vian96/smart_inventory.git
cd smart_inventory
```

2. Создайте и активируйте виртуальное окружение:
```
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для Mac/Linux:
source venv/bin/activate
```

3. Установите зависимости:
```
pip install -r requirements.txt
```

4. Запустите сервер:
```
uvicorn main:app --reload
```

5. Откройте документацию API (Swagger UI): Перейдите в браузере по адресу: http://127.0.0.1:8000/docs

## Тестирование и качество кода

Запуск автоматических тестов с проверкой покрытия:
```
pytest --cov=. --cov-report=term-missing
```

Проверка качества кода линтером Pylint:
```
pylint *.py
ruff check *.py
ruff format *.py
```
